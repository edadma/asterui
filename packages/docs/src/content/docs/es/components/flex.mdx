## API

### `Flex`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `direction` | Dirección flex | `'row' \| 'column' \| 'row-reverse' \| 'column-reverse'` | `'row'` |
| `justify` | Justifica contenido a lo largo del eje principal | `'start' \| 'end' \| 'center' \| 'between' \| 'around' \| 'evenly'` | - |
| `align` | Alinea elementos a lo largo del eje transversal | `'start' \| 'end' \| 'center' \| 'baseline' \| 'stretch'` | - |
| `wrap` | Si ajustar elementos | `boolean \| 'wrap' \| 'nowrap' \| 'wrap-reverse'` | `false` |
| `gap` | Espacio entre elementos | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| number` | - |
| `flex` | Abreviatura flex para el contenedor | `'1' \| 'auto' \| 'initial' \| 'none'` | - |
| `inline` | Usar inline-flex en lugar de flex | `boolean` | `false` |
| `minHeight` | Altura mínima | `'screen' \| 'full' \| 'fit' \| 'min' \| 'max'` | - |
| `minWidth` | Ancho mínimo | `'full' \| 'fit' \| 'min' \| 'max'` | - |
| `className` | Clases CSS adicionales | `string` | - |
| `data-testid` | ID de prueba para pruebas | `string` | - |

## Notas de Uso

- Flex es una utilidad de diseño sin requisitos de accesibilidad específicos
- Mantiene la estructura semántica de los elementos hijos
- Gap se mapea al espaciado de Tailwind: xs=1, sm=2, md=4, lg=6, xl=8
