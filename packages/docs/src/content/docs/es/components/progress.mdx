## API

### `Progress`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `value` | Valor de progreso actual (0-100). Omitir para estado indeterminado | `number` | - |
| `max` | Valor máximo | `number` | `100` |
| `type` | Variante de color | `'neutral' \| 'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | - |
| `className` | Clases CSS adicionales | `string` | - |

## Accesibilidad

- Usa elemento nativo `<progress>` para accesibilidad
- El valor es anunciado por lectores de pantalla
- El estado indeterminado se comunica apropiadamente
