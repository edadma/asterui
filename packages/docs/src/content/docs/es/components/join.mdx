## API

### `Join`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `direction` | Dirección de unión | `'horizontal' \| 'vertical'` | `'horizontal'` |
| `children` | Elementos a unir | `React.ReactNode` | - |
| `className` | Clases CSS adicionales | `string` | - |
