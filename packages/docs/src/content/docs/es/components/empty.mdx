## API

### `Empty`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `description` | Texto o nodo a mostrar como descripción | `React.ReactNode` | `'No Data'` |
| `image` | Imagen o icono personalizado; establece en null para ocultar | `React.ReactNode` | `Empty.PRESENTED_IMAGE_DEFAULT` |
| `imageStyle` | Estilo aplicado al contenedor de imagen | `React.CSSProperties` | - |
| `children` | Contenido adicional como botones o acciones | `React.ReactNode` | - |
| `className` | Clases CSS adicionales | `string` | - |
| `data-testid` | ID de prueba para el componente | `string` | `'empty'` |
