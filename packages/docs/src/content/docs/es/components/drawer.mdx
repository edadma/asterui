## API

### `Drawer`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `children` | Contenido del cajón | `React.ReactNode` | - |
| `open` | Estado abierto controlado | `boolean` | `false` |
| `onClose` | Manejador de cierre | `(e?: React.MouseEvent \| React.KeyboardEvent) => void` | - |
| `afterOpenChange` | Callback después de que se complete la animación de abrir/cerrar | `(open: boolean) => void` | - |
| `title` | Título del cajón | `React.ReactNode` | - |
| `placement` | Posición del cajón | `'left' \| 'right' \| 'top' \| 'bottom'` | `'right'` |
| `size` | Tamaño predefinido (378px o 736px) o número personalizado | `'default' \| 'large' \| number` | `'default'` |
| `width` | Ancho personalizado (anula size para izquierda/derecha) | `string \| number` | - |
| `height` | Altura personalizada (anula size para arriba/abajo) | `string \| number` | - |
| `closable` | Mostrar botón de cierre | `boolean` | `true` |
| `mask` | Mostrar máscara de superposición | `boolean` | `true` |
| `maskClosable` | Cerrar al hacer clic en la máscara | `boolean` | `true` |
| `keyboard` | Cerrar con la tecla ESC | `boolean` | `true` |
| `footer` | Contenido del pie de página | `React.ReactNode` | - |
| `extra` | Contenido adicional en el encabezado (lado derecho) | `React.ReactNode` | - |
| `loading` | Mostrar esqueleto de carga | `boolean` | `false` |
| `destroyOnClose` | Destruir contenido cuando esté cerrado | `boolean` | `false` |
| `forceRender` | Pre-renderizar contenido (mantener en DOM cuando esté cerrado) | `boolean` | `false` |
| `push` | Comportamiento de empuje de cajón anidado | `boolean \| { distance: number }` | `{ distance: 180 }` |
| `zIndex` | z-index del cajón | `number` | `1000` |
| `className` | Clase CSS para el panel del cajón | `string` | - |
