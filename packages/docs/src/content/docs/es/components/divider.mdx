## API

### `Divider`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `children` | Texto o contenido en el divisor | `React.ReactNode` | - |
| `orientation` | Orientación del divisor | `'horizontal' \| 'vertical'` | `'horizontal'` |
| `position` | Posición del texto | `'start' \| 'center' \| 'end'` | `'center'` |
| `type` | Tipo de color | `'neutral' \| 'primary' \| 'secondary' \| 'accent' \| 'success' \| 'warning' \| 'info' \| 'error'` | - |
| `className` | Clases CSS adicionales | `string` | - |
