## API

### `Menu`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `children` | Elementos del menú (patrón compuesto) | `React.ReactNode` | - |
| `items` | Elementos del menú (patrón basado en datos) | `MenuItem[]` | - |
| `mode` | Modo de visualización del menú | `'vertical' \| 'horizontal' \| 'inline'` | `'vertical'` |
| `size` | Tamaño del menú | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | - |
| `selectedKeys` | Claves de elementos seleccionados controlados | `string[]` | - |
| `defaultSelectedKeys` | Claves seleccionadas predeterminadas (no controlado) | `string[]` | `[]` |
| `openKeys` | Claves de submenú abierto controladas | `string[]` | - |
| `defaultOpenKeys` | Claves de submenú abierto predeterminadas (no controlado) | `string[]` | `[]` |
| `onSelect` | Callback cuando se selecciona un elemento | `(key: string) => void` | - |
| `onOpenChange` | Callback cuando cambia el estado abierto del submenú | `(openKeys: string[]) => void` | - |
| `className` | Clases CSS adicionales | `string` | - |

### `MenuItem` (para la prop items)

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `key` | Identificador único | `string` | - |
| `label` | Etiqueta del elemento | `React.ReactNode` | - |
| `icon` | Elemento de icono | `React.ReactNode` | - |
| `disabled` | Si el elemento está deshabilitado | `boolean` | `false` |
| `children` | Elementos del submenú | `MenuItem[]` | - |
| `divider` | Renderizar como divisor | `boolean` | `false` |
| `title` | Renderizar como título de sección | `boolean` | `false` |

### `Menu.Item` (patrón compuesto)

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `key` | Clave única para selección (prop key de React) | `string` | - |
| `children` | Contenido del elemento | `React.ReactNode` | - |
| `icon` | Elemento de icono | `React.ReactNode` | - |
| `disabled` | Si el elemento está deshabilitado | `boolean` | `false` |
| `onClick` | Manejador de clic | `() => void` | - |
| `className` | Clases CSS adicionales | `string` | - |

### `Menu.SubMenu` (patrón compuesto)

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `key` | Clave única para estado abierto (prop key de React) | `string` | - |
| `label` | Etiqueta del submenú | `React.ReactNode` | - |
| `icon` | Elemento de icono | `React.ReactNode` | - |
| `disabled` | Si el submenú está deshabilitado | `boolean` | `false` |
| `children` | Elementos del submenú | `React.ReactNode` | - |
| `className` | Clases CSS adicionales | `string` | - |

### `Menu.Title`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `children` | Contenido del título | `React.ReactNode` | - |
| `className` | Clases CSS adicionales | `string` | - |

### `Menu.Divider`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `className` | Clases CSS adicionales | `string` | - |

## Accesibilidad

- Usa patrones de navegación semánticos con atributos ARIA apropiados
- Soporte de navegación por teclado con teclas de flecha para recorrido del menú
- Los elementos seleccionados están marcados apropiadamente para lectores de pantalla
- Los estados abierto/cerrado de submenú se comunican a tecnologías asistivas
- Los elementos de menú deshabilitados están marcados apropiadamente y no son enfocables
- Los estados de enfoque son visibles para navegación por teclado
