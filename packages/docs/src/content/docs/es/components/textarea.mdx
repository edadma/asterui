## API

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `size` | Tamaño del textarea | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `-` |
| `color` | Tema de color del textarea | `'neutral' \| 'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | `-` |
| `ghost` | Variante de estilo fantasma | `boolean` | `false` |
| `bordered` | Mostrar borde | `boolean` | `true` |
| `disabled` | Estado deshabilitado | `boolean` | `false` |
| `placeholder` | Texto de placeholder | `string` | `-` |
| `rows` | Número de filas de texto visibles | `number` | `-` |
| `className` | Clases CSS adicionales | `string` | `-` |

## Accesibilidad

- Usa `<textarea>` nativo para accesibilidad completa
- Funciona con etiquetas de formulario estándar
- El estado deshabilitado se comunica correctamente
- Redimensionable por defecto para comodidad del usuario
