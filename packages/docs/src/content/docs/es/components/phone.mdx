## API

### `Phone`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `children` | Contenido dentro de la pantalla del teléfono | `React.ReactNode` | - |
| `color` | Color personalizado del marco | `string` | - |
| `className` | Clases para contenedor del teléfono | `string` | - |
| `displayClassName` | Clases para área de visualización | `string` | - |

## Accesibilidad

- El mockup del teléfono es decorativo - el contenido interior debe ser accesible
- Las imágenes dentro deben tener texto `alt` apropiado
- Los elementos interactivos dentro permanecen completamente accesibles
