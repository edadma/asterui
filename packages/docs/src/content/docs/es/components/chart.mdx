Visualización de datos con integración de ApexCharts.

## Instalación

Este componente requiere `apexcharts` como dependencia peer:

```bash
npm install apexcharts
```

## Importar

```tsx
import { Chart } from 'asterui/chart'
```

## API

### `Chart`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `type` | Tipo de gráfico | `'line' \| 'bar' \| 'pie' \| 'donut' \| 'area' \| 'radar' \| 'radialBar'` | `'line'` |
| `series` | Series de datos | `ApexAxisChartSeries \| ApexNonAxisChartSeries` | `[]` |
| `options` | Opciones de ApexCharts | `ApexOptions` | `{}` |
| `height` | Altura del gráfico | `number \| string` | `350` |
| `width` | Ancho del gráfico | `number \| string` | `'100%'` |
| `className` | Clases CSS adicionales | `string` | - |

## Accesibilidad

- Los gráficos incluyen etiquetas ARIA para puntos de datos
- Los elementos de leyenda son navegables por teclado
- Los tooltips proporcionan datos detallados al pasar el mouse/enfocar
- Los colores se eligen para contraste suficiente
