## API

### `Tooltip`

| Propiedad | Descripción | Tipo | Predeterminado |
|----------|-------------|------|---------|
| `tip` | Contenido de texto del tooltip | `string` | `-` |
| `position` | Posición del tooltip | `'top' \| 'bottom' \| 'left' \| 'right'` | `'top'` |
| `color` | Color del tooltip | `'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | `-` |
| `open` | Forzar al tooltip a ser visible | `boolean` | `false` |
| `children` | Elemento que activa el tooltip | `React.ReactNode` | `-` |
| `className` | Clases CSS adicionales | `string` | `-` |

## Accesibilidad

- El contenido del tooltip se expone mediante aria-describedby
- Los tooltips aparecen en focus además de al pasar el cursor
- Contraste de color suficiente en todas las variantes de color
- Las teclas de flecha se pueden usar cuando el tooltip contiene elementos enfocables
