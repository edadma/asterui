## API

### `Join`

| Propriété | Description | Type | Par défaut |
|----------|-------------|------|---------|
| `direction` | Direction de jointure | `'horizontal' \| 'vertical'` | `'horizontal'` |
| `children` | Éléments à joindre ensemble | `React.ReactNode` | - |
| `className` | Classes CSS supplémentaires | `string` | - |
