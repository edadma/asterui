## API

| Propriété | Description | Type | Défaut |
|----------|-------------|------|---------|
| `left` | Contenu du côté gauche (révélé lors du glissement) | `React.ReactNode` | - |
| `right` | Contenu du côté droit (masqué lors du glissement) | `React.ReactNode` | - |
| `aspect` | Classe de rapport d'aspect | `string` | `'aspect-16/9'` |
| `className` | Classes CSS supplémentaires | `string` | - |

## Accessibilité

- Accessible au clavier avec navigation par tabulation
- Utilise `role="img"` pour une signification sémantique
- Les touches fléchées contrôlent la position du redimensionneur lorsque focalisé
