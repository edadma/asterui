## API

### `Empty`

| Propriété | Description | Type | Défaut |
|----------|-------------|------|---------|
| `description` | Texte ou nœud à afficher comme description | `React.ReactNode` | `'No Data'` |
| `image` | Image ou icône personnalisée ; définir à null pour masquer | `React.ReactNode` | `Empty.PRESENTED_IMAGE_DEFAULT` |
| `imageStyle` | Style appliqué au conteneur d'image | `React.CSSProperties` | - |
| `children` | Contenu supplémentaire comme boutons ou actions | `React.ReactNode` | - |
| `className` | Classes CSS supplémentaires | `string` | - |
| `data-testid` | ID de test pour le composant | `string` | `'empty'` |
