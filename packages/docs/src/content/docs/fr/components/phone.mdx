## API

### `Phone`

| Propriété | Description | Type | Défaut |
|----------|-------------|------|---------|
| `children` | Contenu à l'intérieur de l'écran du téléphone | `React.ReactNode` | - |
| `color` | Couleur de cadre personnalisée | `string` | - |
| `className` | Classes pour le conteneur du téléphone | `string` | - |
| `displayClassName` | Classes pour la zone d'affichage | `string` | - |

## Accessibilité

- La maquette de téléphone est décorative - le contenu à l'intérieur doit être accessible
- Les images à l'intérieur doivent avoir un attribut `alt` approprié
- Les éléments interactifs à l'intérieur restent entièrement accessibles
