## API

| Propriété | Description | Type | Défaut |
|----------|-------------|------|---------|
| `type` | Type de couleur du statut | `'neutral' \| 'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | `'neutral'` |
| `size` | Taille du statut | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` |
| `ping` | Effet d'animation ping | `boolean` | `false` |
| `bounce` | Effet d'animation rebond | `boolean` | `false` |
| `label` | Label d'accessibilité | `string` | - |
| `className` | Classes CSS supplémentaires | `string` | - |

## Accessibilité

- Utilisez la propriété `label` pour fournir un nom accessible
- Envisagez d'associer du texte pour les utilisateurs de lecteurs d'écran
- L'animation ping attire l'attention sur les changements de statut importants
