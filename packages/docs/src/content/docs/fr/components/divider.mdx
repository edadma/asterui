## API

### `Divider`

| Propriété | Description | Type | Défaut |
|----------|-------------|------|---------|
| `children` | Texte ou contenu dans le séparateur | `React.ReactNode` | - |
| `orientation` | Orientation du séparateur | `'horizontal' \| 'vertical'` | `'horizontal'` |
| `position` | Position du texte | `'start' \| 'center' \| 'end'` | `'center'` |
| `type` | Type de couleur | `'neutral' \| 'primary' \| 'secondary' \| 'accent' \| 'success' \| 'warning' \| 'info' \| 'error'` | - |
| `className` | Classes CSS supplémentaires | `string` | - |
