## API

### `Transfer`

| Propriété | Description | Type | Défaut |
|----------|-------------|------|---------|
| `dataSource` | Tableau de source de données | `TransferItem[]` | `[]` |
| `targetKeys` | Clés des éléments dans la colonne de droite | `string[]` | `[]` |
| `onChange` | Callback lorsque les éléments sont transférés | `(targetKeys: string[]) => void` | `-` |
| `render` | Fonction de rendu personnalisée pour les éléments | `(item: TransferItem) => React.ReactNode` | `-` |
| `showSearch` | Afficher l'entrée de recherche | `boolean` | `false` |
| `titles` | Titres pour les colonnes de gauche et de droite | `[string, string]` | `['Source', 'Target']` |
| `className` | Classes CSS supplémentaires | `string` | `-` |

### `TransferItem`

| Propriété | Description | Type | Défaut |
|----------|-------------|------|---------|
| `key` | Identifiant unique | `string` | `-` |
| `title` | Titre affiché | `string` | `-` |
| `description` | Description facultative | `string` | `-` |
| `disabled` | Si l'élément est désactivé | `boolean` | `false` |

## Accessibilité

- Utilise des entrées de case à cocher pour la sélection d'éléments
- Navigation au clavier entre les éléments
- Les boutons de transfert sont accessibles au clavier
- Les éléments désactivés sont correctement annoncés
