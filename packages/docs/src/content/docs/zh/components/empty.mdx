## API

### `Empty`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `description` | 作为描述显示的文本或节点 | `React.ReactNode` | `'No Data'` |
| `image` | 自定义图片或图标；设置为 null 隐藏 | `React.ReactNode` | `Empty.PRESENTED_IMAGE_DEFAULT` |
| `imageStyle` | 应用于图片容器的样式 | `React.CSSProperties` | - |
| `children` | 额外内容，如按钮或操作 | `React.ReactNode` | - |
| `className` | 额外的 CSS 类名 | `string` | - |
| `data-testid` | 组件的测试 ID | `string` | `'empty'` |
