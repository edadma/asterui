## API

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `left` | 左侧内容（拖动时显示） | `React.ReactNode` | - |
| `right` | 右侧内容（拖动时隐藏） | `React.ReactNode` | - |
| `aspect` | 宽高比类名 | `string` | `'aspect-16/9'` |
| `className` | 额外的 CSS 类名 | `string` | - |

## 无障碍

- 支持 Tab 键进行键盘导航
- 使用 `role="img"` 表示语义含义
- 聚焦时可用方向键控制调整器位置
