## API

### `Checkbox`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `checked` | 受控的选中状态 | `boolean` | - |
| `defaultChecked` | 默认选中状态 | `boolean` | - |
| `disabled` | 禁用复选框 | `boolean` | `false` |
| `indeterminate` | 不确定状态（部分选择） | `boolean` | `false` |
| `size` | 复选框大小 | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | - |
| `color` | 复选框颜色 | `'primary' \| 'secondary' \| 'accent' \| 'neutral' \| 'success' \| 'warning' \| 'info' \| 'error'` | - |
| `swap` | 交换模式配置 | `CheckboxSwapConfig` | - |
| `value` | 在组中使用时的值 | `string \| number` | - |
| `onChange` | 改变事件处理函数 | `(e: ChangeEvent) => void` | - |
| `name` | 输入的 HTML name 属性 | `string` | - |
| `autoFocus` | 挂载时自动聚焦 | `boolean` | `false` |
| `className` | 额外的 CSS 类名 | `string` | - |
| `children` | 标签内容（自动包裹在 label 元素中） | `ReactNode` | - |

### `Checkbox.Group`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `value` | 受控的选中值 | `(string \| number)[]` | - |
| `defaultValue` | 默认选中值 | `(string \| number)[]` | - |
| `onChange` | 选择改变回调函数 | `(values: (string \| number)[]) => void` | - |
| `disabled` | 禁用所有复选框 | `boolean` | `false` |
| `options` | 复选框选项 | `(string \| number \| CheckboxOptionType)[]` | - |
| `direction` | 布局方向 | `'horizontal' \| 'vertical'` | `'vertical'` |
| `name` | 所有复选框的 HTML name 属性（用于表单提交） | `string` | - |
| `children` | 手动复选框元素 | `ReactNode` | - |

### `CheckboxSwapConfig`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `on` | 选中时显示的内容 | `ReactNode` | - |
| `off` | 未选中时显示的内容 | `ReactNode` | - |
| `effect` | 动画效果 | `'rotate' \| 'flip'` | - |

### `CheckboxOptionType`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `label` | 选项标签 | `ReactNode` | - |
| `value` | 选项值 | `string \| number` | - |
| `disabled` | 禁用此选项 | `boolean` | `false` |

## 无障碍

- 使用原生复选框输入以获得键盘和屏幕阅读器支持
- 正确地将标签与复选框关联
- 不确定状态通过原生 indeterminate 属性传达
- 禁用状态防止交互并传达给辅助技术
- 焦点状态对键盘导航可见
