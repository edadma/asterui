## API

### `Footer`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `children` | 页脚内容 | `React.ReactNode` | - |
| `center` | 居中页脚内容 | `boolean` | `false` |
| `horizontal` | 水平显示列 | `boolean` | `false` |
| `vertical` | 垂直显示列（默认） | `boolean` | `false` |
| `className` | 额外的 CSS 类名 | `string` | - |

### `Footer.Title`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `children` | 标题文本 | `React.ReactNode` | - |
| `className` | 额外的 CSS 类名 | `string` | - |
