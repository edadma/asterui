## API

### `Container`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `children` | 容器内容 | `React.ReactNode` | - |
| `size` | 最大宽度尺寸 | `'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl' \| 'full'` | `'xl'` |
| `centered` | 水平居中容器 | `boolean` | `true` |
| `padding` | 添加横向内边距 | `boolean` | `true` |
| `className` | 额外的 CSS 类名 | `string` | - |
