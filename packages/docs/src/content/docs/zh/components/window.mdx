## API

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `children` | 窗口内容 | `React.ReactNode` | - |
| `className` | 窗口的额外 CSS 类 | `string` | - |
| `contentClassName` | 内容区域的额外 CSS 类 | `string` | - |

## 可访问性

- 窗口框架是装饰性的,标记为表现性
- 内容区域保持正确的阅读顺序
- 代码块可由屏幕阅读器读取
- 交通灯按钮是非交互式装饰
