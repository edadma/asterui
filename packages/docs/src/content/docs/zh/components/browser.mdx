## API

### `Browser`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `children` | 浏览器视口内的内容 | `React.ReactNode` | - |
| `url` | 地址栏中显示的 URL | `string` | `'https://example.com'` |
| `className` | 浏览器容器的类名 | `string` | - |
| `contentClassName` | 内容区域的类名 | `string` | - |
