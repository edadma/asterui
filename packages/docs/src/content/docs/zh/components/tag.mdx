## API

### `Tag`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `closable` | 显示关闭图标并启用关闭 | `boolean` | `false` |
| `closeIcon` | 自定义关闭图标元素 | `ReactNode` | `-` |
| `onClose` | 标签关闭时的回调 | `() => void` | `-` |
| `color` | 标签颜色(预设或自定义十六进制) | `'primary' \| 'secondary' \| 'accent' \| 'neutral' \| 'info' \| 'success' \| 'warning' \| 'error' \| 'ghost' \| string` | `-` |
| `icon` | 文本前显示的图标元素 | `ReactNode` | `-` |
| `size` | 标签大小 | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` |
| `variant` | 标签样式变体 | `'filled' \| 'outlined' \| 'soft' \| 'dash'` | `'filled'` |
| `visible` | 受控可见性状态 | `boolean` | `-` |
| `disabled` | 禁用交互 | `boolean` | `false` |
| `href` | 将标签渲染为链接 | `string` | `-` |
| `target` | 链接目标(设置 href 时) | `string` | `-` |
| `children` | 标签内容 | `ReactNode` | `-` |
| `className` | 额外的 CSS 类 | `string` | `-` |
| `data-testid` | 组件的测试 ID | `string` | `'tag'` |
| `aria-label` | 可访问标签(在关闭按钮公告中使用) | `string` | `-` |

### `CheckableTag`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `checked` | 标签是否勾选 | `boolean` | `false` |
| `onChange` | 勾选状态更改时的回调 | `(checked: boolean) => void` | `-` |
| `icon` | 文本前显示的图标元素 | `ReactNode` | `-` |
| `size` | 标签大小 | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` |
| `color` | 勾选时的标签颜色 | `'primary' \| 'secondary' \| 'accent' \| 'neutral' \| 'info' \| 'success' \| 'warning' \| 'error' \| 'ghost'` | `'primary'` |
| `disabled` | 禁用交互 | `boolean` | `false` |
| `children` | 标签内容 | `ReactNode` | `-` |
| `className` | 额外的 CSS 类 | `string` | `-` |
| `data-testid` | 组件的测试 ID | `string` | `'checkable-tag'` |

## 可访问性

- 关闭按钮包含描述性的 `aria-label`(例如,"删除标签 1")
- CheckableTag 使用带有正确 `aria-checked` 状态的 `role="checkbox"`
- 可通过键盘访问 - 关闭和切换操作使用 Enter/Space 工作
- 所有交互元素上的可见焦点指示器
- `TagLiveRegion` 向屏幕阅读器宣布标签移除
- 禁用时设置 `aria-disabled` 属性
- 预设颜色的颜色对比度符合 WCAG 指南
