## API

### `Mention`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `options` | 提及选项数组 | `Array<{ value: string, label: string }>` | - |
| `value` | 受控值 | `string` | - |
| `onChange` | 值改变时的回调 | `(value: string) => void` | - |
| `prefix` | 触发提及建议的字符 | `string` | `'@'` |
| `split` | 用于在输入中分隔提及的字符 | `string` | - |
| `placeholder` | 输入占位符文本 | `string` | - |
| `disabled` | 禁用提及输入 | `boolean` | `false` |
| `className` | 额外的 CSS 类名 | `string` | - |

## 键盘导航

- **↑** / **↓** - 浏览提及建议
- **Enter** - 选择高亮显示的提及
- **Esc** - 关闭提及下拉菜单

## 无障碍

- 完整的键盘导航支持以选择提及
- ARIA 属性，供屏幕阅读器朗读建议
- 下拉菜单打开和关闭时的焦点管理
- 选中和聚焦项目的清晰视觉指示器
