## API

### `Kbd`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `size` | kbd 的尺寸 | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | - |
| `children` | 按键内容 | `React.ReactNode` | - |
| `className` | 额外的 CSS 类名 | `string` | - |

## 无障碍

- 使用语义化的 `<kbd>` HTML 元素
- 屏幕阅读器适当地朗读键盘按键
- 高对比度样式确保可见性
