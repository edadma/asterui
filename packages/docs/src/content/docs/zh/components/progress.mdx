## API

### `Progress`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `value` | 当前进度值(0-100)。省略以获得不确定状态 | `number` | - |
| `max` | 最大值 | `number` | `100` |
| `type` | 颜色变体 | `'neutral' \| 'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | - |
| `className` | 额外的 CSS 类 | `string` | - |

## 可访问性

- 使用原生 `<progress>` 元素以获得可访问性
- 值由屏幕阅读器宣布
- 不确定状态得到正确传达
