## API

### `Flex`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `direction` | 弹性方向 | `'row' \| 'column' \| 'row-reverse' \| 'column-reverse'` | `'row'` |
| `justify` | 沿主轴对齐内容 | `'start' \| 'end' \| 'center' \| 'between' \| 'around' \| 'evenly'` | - |
| `align` | 沿交叉轴对齐项目 | `'start' \| 'end' \| 'center' \| 'baseline' \| 'stretch'` | - |
| `wrap` | 是否换行项目 | `boolean \| 'wrap' \| 'nowrap' \| 'wrap-reverse'` | `false` |
| `gap` | 项目之间的间距 | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| number` | - |
| `flex` | 容器的弹性简写 | `'1' \| 'auto' \| 'initial' \| 'none'` | - |
| `inline` | 使用 inline-flex 代替 flex | `boolean` | `false` |
| `minHeight` | 最小高度 | `'screen' \| 'full' \| 'fit' \| 'min' \| 'max'` | - |
| `minWidth` | 最小宽度 | `'full' \| 'fit' \| 'min' \| 'max'` | - |
| `className` | 额外的 CSS 类名 | `string` | - |

## 使用说明

- Flex 是一个布局工具，没有特定的无障碍要求
- 保持子元素的语义结构
- Gap 映射到 Tailwind 间距：xs=1, sm=2, md=4, lg=6, xl=8
