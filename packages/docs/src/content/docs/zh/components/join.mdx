## API

### `Join`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `direction` | 连接方向 | `'horizontal' \| 'vertical'` | `'horizontal'` |
| `children` | 要连接在一起的元素 | `React.ReactNode` | - |
| `className` | 额外的 CSS 类名 | `string` | - |
