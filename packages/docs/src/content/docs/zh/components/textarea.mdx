## API

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `size` | 文本域大小 | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `-` |
| `color` | 文本域颜色主题 | `'neutral' \| 'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | `-` |
| `ghost` | 幽灵样式变体 | `boolean` | `false` |
| `bordered` | 显示边框 | `boolean` | `true` |
| `disabled` | 禁用状态 | `boolean` | `false` |
| `placeholder` | 占位符文本 | `string` | `-` |
| `rows` | 可见文本行数 | `number` | `-` |
| `className` | 额外的 CSS 类 | `string` | `-` |
| `data-testid` | 用于测试的测试 ID | `string` | - |

## 可访问性

- 使用原生 `<textarea>` 以获得完全的可访问性
- 与标准表单标签配合使用
- 禁用状态得到正确传达
- 默认情况下可调整大小以提高用户舒适度
