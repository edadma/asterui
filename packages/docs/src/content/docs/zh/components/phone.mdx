## API

### `Phone`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `children` | 手机屏幕内的内容 | `React.ReactNode` | - |
| `color` | 自定义框架颜色 | `string` | - |
| `className` | 手机容器的类 | `string` | - |
| `displayClassName` | 显示区域的类 | `string` | - |

## 可访问性

- 手机模型是装饰性的 - 内部内容应该是可访问的
- 内部的图片应该有适当的 `alt` 文本
- 内部的交互元素保持完全可访问
