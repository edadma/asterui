## API

### `Tooltip`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `tip` | 工具提示文本内容 | `string` | `-` |
| `position` | 工具提示位置 | `'top' \| 'bottom' \| 'left' \| 'right'` | `'top'` |
| `color` | 工具提示颜色 | `'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | `-` |
| `open` | 强制工具提示可见 | `boolean` | `false` |
| `children` | 触发工具提示的元素 | `React.ReactNode` | `-` |
| `className` | 额外的 CSS 类 | `string` | `-` |

## 可访问性

- 工具提示内容通过 aria-describedby 公开
- 工具提示在焦点和悬停时都会出现
- 所有颜色变体都有足够的颜色对比度
- 当工具提示包含可聚焦元素时可以使用箭头键
