## API

### `Fieldset`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `children` | fieldset 的内容 | `React.ReactNode` | - |
| `disabled` | 禁用内部所有表单控件 | `boolean` | `false` |
| `className` | 额外的 CSS 类名 | `string` | - |

### `Fieldset.Legend`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `children` | 图例文本或内容 | `React.ReactNode` | - |
| `className` | 额外的 CSS 类名 | `string` | - |

### `Fieldset.Label`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `children` | 标签文本或内容 | `React.ReactNode` | - |
| `htmlFor` | 标签所针对的表单元素的 ID | `string` | - |
| `className` | 额外的 CSS 类名 | `string` | - |
