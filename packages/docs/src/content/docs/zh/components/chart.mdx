## 安装

此组件需要 `apexcharts` 作为对等依赖项：

```bash
npm install apexcharts
```

## 引入

```tsx
import { Chart } from 'asterui/chart'
```

## API

### `Chart`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `type` | 图表类型 | `'line' \| 'bar' \| 'pie' \| 'donut' \| 'area' \| 'radar' \| 'radialBar'` | `'line'` |
| `series` | 数据系列 | `ApexAxisChartSeries \| ApexNonAxisChartSeries` | `[]` |
| `options` | ApexCharts 选项 | `ApexOptions` | `{}` |
| `height` | 图表高度 | `number \| string` | `350` |
| `width` | 图表宽度 | `number \| string` | `'100%'` |
| `className` | 额外的 CSS 类名 | `string` | - |

## 无障碍

- 图表包含数据点的 ARIA 标签
- 图例项可通过键盘导航
- 工具提示在悬停/焦点时提供详细数据
- 选择的颜色具有足够的对比度
