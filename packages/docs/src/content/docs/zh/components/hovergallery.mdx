## API

### `HoverGallery`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `images` | 图片 URL 数组（2-10 张图片） | `string[]` | - |
| `alts` | 每张图片的替代文本 | `string[]` | - |
| `className` | 额外的 CSS 类名 | `string` | - |

## 注意

- 支持最多 10 张图片
- 默认显示第一张图片
- 纯 CSS - 悬停效果不需要 JavaScript
- 最好使用相同宽高比的图片
