## API

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `items` | 要轮播的文本项目(最多 6 个) | `React.ReactNode[]` | - |
| `duration` | 动画持续时间(毫秒) | `6000 \| 9000 \| 12000 \| 15000 \| 18000` | `10000` |
| `centered` | 水平居中文本 | `boolean` | `false` |
| `className` | 额外的 CSS 类 | `string` | - |
| `data-testid` | 用于测试的测试 ID | `string` | - |

## 可访问性

- 悬停时动画暂停,以便更轻松地阅读
- 支持减少动作偏好
- 文本在所有动画状态下都保持可读
