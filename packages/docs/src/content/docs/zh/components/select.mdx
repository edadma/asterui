## API

### `Select`

| 属性 | 描述 | 类型 | 默认值 |
|----------|-------------|------|---------|
| `size` | 选择器大小 | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | - |
| `color` | 选择器颜色主题 | `'neutral' \| 'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | - |
| `ghost` | 幽灵样式变体 | `boolean` | `false` |
| `bordered` | 显示边框 | `boolean` | `true` |
| `disabled` | 禁用状态 | `boolean` | `false` |
| `className` | 额外的 CSS 类 | `string` | - |
| `children` | 选项元素 | `React.ReactNode` | - |
| `data-testid` | 用于测试的测试 ID | `string` | - |

## 可访问性

- 使用原生 `<select>` 以获得完全的可访问性
- 使用箭头键和 Enter 进行键盘导航
- 与标准表单标签配合使用
- 禁用状态得到正确传达
