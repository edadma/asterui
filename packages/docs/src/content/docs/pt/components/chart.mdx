Visualização de dados com integração do ApexCharts.

## Instalação

Este componente requer `apexcharts` como dependência peer:

```bash
npm install apexcharts
```

## Importação

```tsx
import { Chart } from 'asterui/chart'
```

## API

### `Chart`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `type` | Tipo de gráfico | `'line' \| 'bar' \| 'pie' \| 'donut' \| 'area' \| 'radar' \| 'radialBar'` | `'line'` |
| `series` | Série de dados | `ApexAxisChartSeries \| ApexNonAxisChartSeries` | `[]` |
| `options` | Opções do ApexCharts | `ApexOptions` | `{}` |
| `height` | Altura do gráfico | `number \| string` | `350` |
| `width` | Largura do gráfico | `number \| string` | `'100%'` |
| `className` | Classes CSS adicionais | `string` | - |

## Acessibilidade

- Gráficos incluem rótulos ARIA para pontos de dados
- Itens de legenda são navegáveis por teclado
- Tooltips fornecem dados detalhados ao passar o mouse/focar
- Cores são escolhidas para contraste suficiente
