## API

### `Checkbox`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `checked` | Estado marcado controlado | `boolean` | - |
| `defaultChecked` | Estado marcado padrão | `boolean` | - |
| `disabled` | Desabilitar o checkbox | `boolean` | `false` |
| `indeterminate` | Estado indeterminado (seleção parcial) | `boolean` | `false` |
| `size` | Tamanho do checkbox | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | - |
| `color` | Cor do checkbox | `'primary' \| 'secondary' \| 'accent' \| 'neutral' \| 'success' \| 'warning' \| 'info' \| 'error'` | - |
| `swap` | Configuração do modo swap | `CheckboxSwapConfig` | - |
| `value` | Valor quando usado em um grupo | `string \| number` | - |
| `onChange` | Handler de evento de mudança | `(e: ChangeEvent) => void` | - |
| `name` | Atributo name HTML para o input | `string` | - |
| `autoFocus` | Foco automático ao montar | `boolean` | `false` |
| `className` | Classes CSS adicionais | `string` | - |
| `children` | Conteúdo do rótulo (automaticamente envolvido em elemento label) | `ReactNode` | - |

### `Checkbox.Group`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `value` | Valores selecionados controlados | `(string \| number)[]` | - |
| `defaultValue` | Valores selecionados padrão | `(string \| number)[]` | - |
| `onChange` | Callback de mudança de seleção | `(values: (string \| number)[]) => void` | - |
| `disabled` | Desabilitar todos os checkboxes | `boolean` | `false` |
| `options` | Opções de checkbox | `(string \| number \| CheckboxOptionType)[]` | - |
| `direction` | Direção do layout | `'horizontal' \| 'vertical'` | `'vertical'` |
| `name` | Atributo name HTML para todos os checkboxes (para envio de formulário) | `string` | - |
| `children` | Elementos de checkbox manuais | `ReactNode` | - |

### `CheckboxSwapConfig`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `on` | Conteúdo mostrado quando marcado | `ReactNode` | - |
| `off` | Conteúdo mostrado quando desmarcado | `ReactNode` | - |
| `effect` | Efeito de animação | `'rotate' \| 'flip'` | - |

### `CheckboxOptionType`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `label` | Rótulo da opção | `ReactNode` | - |
| `value` | Valor da opção | `string \| number` | - |
| `disabled` | Desabilitar esta opção | `boolean` | `false` |

## Acessibilidade

- Usa input de checkbox nativo para suporte de teclado e leitor de tela
- Associa adequadamente rótulos com checkboxes
- Estado indeterminado é comunicado via propriedade indeterminate nativa
- Estado desabilitado previne interação e é comunicado para tecnologias assistivas
- Estados de foco são visíveis para navegação por teclado
