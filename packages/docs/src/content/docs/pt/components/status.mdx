## API

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `type` | Tipo de cor do status | `'neutral' \| 'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | `'neutral'` |
| `size` | Tamanho do status | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` |
| `ping` | Efeito de animação ping | `boolean` | `false` |
| `bounce` | Efeito de animação bounce | `boolean` | `false` |
| `label` | Label de acessibilidade | `string` | - |
| `className` | Classes CSS adicionais | `string` | - |

## Acessibilidade

- Use a prop `label` para fornecer um nome acessível
- Considere parear com texto para usuários de leitores de tela
- Animação ping chama atenção para mudanças de status importantes
