## API

### `Join`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `direction` | Direção do join | `'horizontal' \| 'vertical'` | `'horizontal'` |
| `children` | Elementos para unir | `React.ReactNode` | - |
| `className` | Classes CSS adicionais | `string` | - |
