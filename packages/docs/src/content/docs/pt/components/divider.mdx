## API

### `Divider`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `children` | Texto ou conteúdo no divisor | `React.ReactNode` | - |
| `orientation` | Orientação do divisor | `'horizontal' \| 'vertical'` | `'horizontal'` |
| `position` | Posição do texto | `'start' \| 'center' \| 'end'` | `'center'` |
| `type` | Tipo de cor | `'neutral' \| 'primary' \| 'secondary' \| 'accent' \| 'success' \| 'warning' \| 'info' \| 'error'` | - |
| `className` | Classes CSS adicionais | `string` | - |
