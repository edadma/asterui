## API

### `Popover`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `children` | Elemento gatilho | `React.ReactNode` | - |
| `content` | Conteúdo do popover | `React.ReactNode` | - |
| `title` | Título do popover | `React.ReactNode` | - |
| `trigger` | Como o popover é acionado | `'hover' \| 'click' \| 'focus'` | `'hover'` |
| `placement` | Posição do popover | `'top' \| 'topLeft' \| 'topRight' \| 'bottom' \| 'bottomLeft' \| 'bottomRight' \| 'left' \| 'leftTop' \| 'leftBottom' \| 'right' \| 'rightTop' \| 'rightBottom'` | `'top'` |
| `open` | Se o popover está visível (modo controlado) | `boolean` | - |
| `onOpenChange` | Callback quando visibilidade muda | `(open: boolean) => void` | - |
| `className` | Classes CSS adicionais para wrapper | `string` | - |
| `overlayClassName` | Classes CSS adicionais para overlay do popover | `string` | - |

## Acessibilidade

- Popover usa atributos ARIA apropriados para tooltips
- Gatilhos de clique e foco são acessíveis por teclado
- Conteúdo é anunciado aos leitores de tela quando mostrado
- Tecla Escape fecha popovers acionados por clique
