## API

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `size` | Tamanho do textarea | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `-` |
| `color` | Tema de cor do textarea | `'neutral' \| 'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | `-` |
| `ghost` | Variante de estilo fantasma | `boolean` | `false` |
| `bordered` | Mostrar borda | `boolean` | `true` |
| `disabled` | Estado desabilitado | `boolean` | `false` |
| `placeholder` | Texto de placeholder | `string` | `-` |
| `rows` | Número de linhas de texto visíveis | `number` | `-` |
| `className` | Classes CSS adicionais | `string` | `-` |
| `data-testid` | ID de teste para testes | `string` | - |

## Acessibilidade

- Usa `<textarea>` nativo para acessibilidade completa
- Funciona com labels de formulário padrão
- Estado desabilitado é comunicado corretamente
- Redimensionável por padrão para conforto do usuário
