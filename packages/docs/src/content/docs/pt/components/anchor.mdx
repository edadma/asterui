## API

### `Anchor`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `items` | Configuração dos links de âncora | `AnchorLinkItem[]` | - |
| `direction` | Direção do layout | `'horizontal' \| 'vertical'` | `'vertical'` |
| `offsetTop` | Deslocamento do topo ao calcular a posição de rolagem | `number` | `0` |
| `bounds` | Distância limite da área de âncora | `number` | `5` |
| `getContainer` | Contêiner de rolagem (padrão: window) | `() => HTMLElement \| Window` | - |
| `getCurrentAnchor` | Personalizar o destaque de âncora | `(activeLink: string) => string` | - |
| `activeLink` | Link atualmente ativo (controlado) | `string` | - |
| `onChange` | Callback quando o link ativo muda | `(activeLink: string) => void` | - |
| `onClick` | Callback quando um link é clicado | `(e, link) => void` | - |
| `affix` | Fixar âncora durante a rolagem | `boolean` | `false` |
| `affixOffsetTop` | Deslocamento de pixels do topo quando affix é true | `number` | `0` |
| `replace` | Substituir histórico em vez de adicionar | `boolean` | `false` |
| `className` | Classe CSS personalizada | `string` | - |
| `children` | Filhos Anchor.Link (alternativa a items) | `React.ReactNode` | - |

### `AnchorLinkItem`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `href` | ID do elemento alvo (sem #) | `string` | - |
| `title` | Título do link | `React.ReactNode` | - |
| `children` | Itens de link aninhados | `AnchorLinkItem[]` | - |

## Acessibilidade

- Usa o elemento semântico `<nav>` para navegação
- Links são acessíveis via teclado
- O estado ativo é indicado visualmente
- A rolagem suave respeita as preferências de movimento do usuário
