## API

### `Empty`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `description` | Texto ou nó para exibir como descrição | `React.ReactNode` | `'No Data'` |
| `image` | Imagem ou ícone personalizado; defina como null para ocultar | `React.ReactNode` | `Empty.PRESENTED_IMAGE_DEFAULT` |
| `imageStyle` | Estilo aplicado ao container da imagem | `React.CSSProperties` | - |
| `children` | Conteúdo extra como botões ou ações | `React.ReactNode` | - |
| `className` | Classes CSS adicionais | `string` | - |
| `data-testid` | ID de teste para o componente | `string` | `'empty'` |
