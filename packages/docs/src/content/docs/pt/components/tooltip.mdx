## API

### `Tooltip`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `tip` | Conteúdo de texto do tooltip | `string` | `-` |
| `position` | Posição do tooltip | `'top' \| 'bottom' \| 'left' \| 'right'` | `'top'` |
| `color` | Cor do tooltip | `'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | `-` |
| `open` | Forçar tooltip a ser visível | `boolean` | `false` |
| `children` | Elemento que ativa o tooltip | `React.ReactNode` | `-` |
| `className` | Classes CSS adicionais | `string` | `-` |

## Acessibilidade

- Conteúdo do tooltip é exposto via aria-describedby
- Tooltips aparecem no foco assim como ao passar o cursor
- Contraste de cor suficiente em todas as variantes de cor
- Teclas de seta podem ser usadas quando tooltip contém elementos focáveis
