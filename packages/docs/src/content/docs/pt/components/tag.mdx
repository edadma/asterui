## API

### `Tag`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `closable` | Mostrar ícone de fechar e habilitar fechamento | `boolean` | `false` |
| `closeIcon` | Elemento de ícone de fechar personalizado | `ReactNode` | `-` |
| `onClose` | Callback quando a tag é fechada | `() => void` | `-` |
| `color` | Cor da tag (predefinida ou hex personalizado) | `'primary' \| 'secondary' \| 'accent' \| 'neutral' \| 'info' \| 'success' \| 'warning' \| 'error' \| 'ghost' \| string` | `-` |
| `icon` | Elemento de ícone a exibir antes do texto | `ReactNode` | `-` |
| `size` | Tamanho da tag | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` |
| `variant` | Variante de estilo da tag | `'filled' \| 'outlined' \| 'soft' \| 'dash'` | `'filled'` |
| `visible` | Estado de visibilidade controlado | `boolean` | `-` |
| `disabled` | Desabilitar interações | `boolean` | `false` |
| `href` | Renderizar tag como link | `string` | `-` |
| `target` | Destino do link (quando href está definido) | `string` | `-` |
| `children` | Conteúdo da tag | `ReactNode` | `-` |
| `className` | Classes CSS adicionais | `string` | `-` |
| `data-testid` | ID de teste para o componente | `string` | `'tag'` |
| `aria-label` | Label acessível (usado no anúncio do botão fechar) | `string` | `-` |

### `CheckableTag`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `checked` | Se a tag está marcada | `boolean` | `false` |
| `onChange` | Callback quando o estado marcado muda | `(checked: boolean) => void` | `-` |
| `icon` | Elemento de ícone a exibir antes do texto | `ReactNode` | `-` |
| `size` | Tamanho da tag | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` |
| `color` | Cor da tag quando marcada | `'primary' \| 'secondary' \| 'accent' \| 'neutral' \| 'info' \| 'success' \| 'warning' \| 'error' \| 'ghost'` | `'primary'` |
| `disabled` | Desabilitar interações | `boolean` | `false` |
| `children` | Conteúdo da tag | `ReactNode` | `-` |
| `className` | Classes CSS adicionais | `string` | `-` |
| `data-testid` | ID de teste para o componente | `string` | `'checkable-tag'` |

## Acessibilidade

- Botão de fechar inclui `aria-label` descritivo (ex. "Remover Tag 1")
- CheckableTag usa `role="checkbox"` com estado `aria-checked` adequado
- Acessível por teclado - ações de fechar e alternar funcionam com Enter/Space
- Indicadores de foco visíveis em todos os elementos interativos
- `TagLiveRegion` anuncia remoções de tags para leitores de tela
- Atributo `aria-disabled` definido quando desabilitado
- Contraste de cor atende às diretrizes WCAG para cores predefinidas
