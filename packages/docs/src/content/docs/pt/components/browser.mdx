## API

### `Browser`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `children` | Conteúdo dentro da janela de visualização do navegador | `React.ReactNode` | - |
| `url` | URL mostrada na barra de endereço | `string` | `'https://example.com'` |
| `className` | Classes para o contêiner do navegador | `string` | - |
| `contentClassName` | Classes para a área de conteúdo | `string` | - |
