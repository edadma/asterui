## Instalação

Este componente requer `@tanstack/react-virtual` como dependência peer:

```bash
npm install @tanstack/react-virtual
```

## Importação

```tsx
import { VirtualList } from 'asterui/virtuallist'
```

## API

### `VirtualList`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `items` | Array de itens a renderizar | `T[]` | - |
| `height` | Altura do container com scroll | `number \| string` | - |
| `itemHeight` | Altura fixa, ou função retornando altura estimada por item para alturas variáveis | `number \| ((item: T, index: number) => number)` | - |
| `renderItem` | Função de renderização para cada item | `(item: T, index: number) => ReactNode` | - |
| `overscan` | Número de itens a renderizar fora da área visível | `number` | `5` |
| `gap` | Espaço entre itens em pixels | `number` | `0` |
| `width` | Largura do container | `number \| string` | - |
| `className` | Classe adicional para o container de scroll | `string` | - |
| `innerClassName` | Classe adicional para o container interno | `string` | - |
| `itemClassName` | Classe adicional para cada wrapper de item | `string` | - |
| `onScroll` | Callback quando posição de scroll muda | `(scrollTop: number) => void` | - |

## Dicas de Performance

- Use um `itemHeight` fixo quando possível para melhor performance
- Memoize sua função `renderItem` se for cara
- Aumente `overscan` se ver flickering durante scroll rápido
- Mantenha componentes de itens simples - evite cálculos pesados no render
