## API

### `Flex`

| Propriedade | Descrição | Tipo | Padrão |
|----------|-------------|------|---------|
| `direction` | Direção do flex | `'row' \| 'column' \| 'row-reverse' \| 'column-reverse'` | `'row'` |
| `justify` | Justify content ao longo do eixo principal | `'start' \| 'end' \| 'center' \| 'between' \| 'around' \| 'evenly'` | - |
| `align` | Alinhar itens ao longo do eixo transversal | `'start' \| 'end' \| 'center' \| 'baseline' \| 'stretch'` | - |
| `wrap` | Se deve quebrar itens | `boolean \| 'wrap' \| 'nowrap' \| 'wrap-reverse'` | `false` |
| `gap` | Espaço entre itens | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| number` | - |
| `flex` | Abreviação flex para o container | `'1' \| 'auto' \| 'initial' \| 'none'` | - |
| `inline` | Usar inline-flex em vez de flex | `boolean` | `false` |
| `minHeight` | Altura mínima | `'screen' \| 'full' \| 'fit' \| 'min' \| 'max'` | - |
| `minWidth` | Largura mínima | `'full' \| 'fit' \| 'min' \| 'max'` | - |
| `className` | Classes CSS adicionais | `string` | - |

## Notas de Uso

- Flex é um utilitário de layout sem requisitos específicos de acessibilidade
- Mantém a estrutura semântica dos elementos filhos
- Gap mapeia para espaçamento Tailwind: xs=1, sm=2, md=4, lg=6, xl=8
